# MAPPING-CRIME-TRENDS

<p>
This project analyzes crime data across U.S. cities and states to identify patterns and trends. The goal is to provide actionable insights for law enforcement and community stakeholders by examining crime frequency, locations, and temporal patterns. The analysis includes descriptive statistics, data aggregation, and visualization using R.
</p>

## Features

- Calculated state-level descriptive statistics, including mean (7.79) and median (7.25) murder rates, to identify high-risk areas.
- Filtered and aggregated city-level crime data to highlight hotspots such as New York City and Chicago.
- Examined city-specific trends, e.g., identifying peak months, highest-incident addresses, and most frequent crime types in Fargo.
- Generated visualizations in R to communicate insights effectively.

## Tools & Technologies

- R for statistical calculations, data cleaning, aggregation, and visualization.

## Insights

- Identified cities and addresses with high crime incidence to support targeted interventions.
- Highlighted temporal trends in crime to inform resource allocation and policy decisions.
- Provided a clear, data-driven understanding of crime patterns for stakeholders.
